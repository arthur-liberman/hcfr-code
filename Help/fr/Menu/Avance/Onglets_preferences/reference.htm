Colorimètre HCFR

Référence...
 

Cette boîte permet de définir les valeurs de référence utilisées lors des calculs colorimétriques.
 

En cliquant sur l'onglet "Références" , la boite de dialogue se présente ainsi.
 

 
 

Plusieurs options de disponibles:
 

  • Couleurs

 

      1. Standard 

        L        a combobox Standard permet de définir la norme utilisée. Cela détermine la norme du gamut (position des couleurs primaires sur le diagramme CIE).

        Trois options possibles :

        PAL/SECAM         
        Le format PAL/SECAM (Phase Alternating Line/Séquentiel Couleur avec Mémoire), utilisé en Europe pour la télévision hertzienne.

        SDTV - Rec601 (NTSC)
        NTSC         (National Television System Committee, c'est-à-dire Comité du système de télévision national) est une norme de codage de la vidéo en couleur mise au point aux USA dans les années 1950. C'est la norme à privilégier pour diffuser du contenu SD.

        HDTV - Rec709         
        C'est la norme retenue pour diffuser du contenu HD

        sRGB         
        C'est la norme utilisée pour diffuser des photos numériques.

         

      2. Blanc

        Illuminant D65
        La combobox Blanc permet de définir le blanc de référence. C’est à partir de ce blanc que sont calculés les écarts dans l’histogramme RVB et le Delta E, mais aussi les valeurs affichées dans les contrôles Niveaux RVB et Cible. Le blanc dépend du standard colorimétrique choisi, mais il est possible de le modifier en cochant la checkbox "Changer le blanc".

         

 

  • Calcul du gamma

 

Cette option permet de définir de quelle manière le gamma moyen est calculé.

 

      1. Standard de visualisation

      2. Standard avec compensation du noir

      3. Standard d'encodage

      4. Standard d'encodage, offset manuel

      5. Optimisé



 

    • Référence Gamma

  

La Référence Gamma permet de déterminer la courbe de référence du gamma dans le graphe de luminance.
