CIE Diagram

The "Display background" option in contextual menu allow displaying the background representing colorspace in visible spectrum.
 

 

This background is of course in false colors, because it represents what our eyes can see (which is more than a projector can display). In this background appears the reference gamut triangle, which depends on reference colorspace, defined in Preferences / References. The triangle points represent red, green and blue components. Colors inside this triangle are those which can be reproduced with respect to the reference colorspace. The black body curve is represented on this background.
 

 
The option "Display Delta E = 3 et 10" allow surrounding with a blue line the zones corresponding to a delta E of 3 and 10 around reference white. Those two blue ovals are only pedagogical, and let you visualize precision necessary to a precise calibration.

 

 

 

Here are in zoom mode the two zones displayed around D65 point.

 

 

The option "Display references" allow displaying or not reference points Illuminant A, B, C, D65 and some usual color temperatures, visible in above image.
